فۆرمی هەڵبژاردنی بەش

بۆ قۆناغی سێیەمی خوێندن

ڕێگای هۆستکردنی فۆرمەکە:

  1. کۆدی سەرەوە کۆپی بکە و لە فایلێکی HTML هەڵیبگریە (وەک: form.html)
  2. فایلەکە هۆست بکە لەسەر هەر سێرڤەرێکی وێب (وەک: GitHub Pages, Netlify, Vercel)
  3. بۆ هۆستکردنی ئاسان لەسەر GitHub Pages:
    • هەژمارێکی GitHub دروست بکە (ئەگەر نییە)
    • Repositoryێکی نوێ دروست بکە
    • فایلەکە Upload بکە و ناوی index.html بنێ
    • بچەرە سەر Settings > Pages و Branchی main هەڵبژێرە
    • لە ماوەی چەند خولەکێکدا فۆرمەکەت دەردەکەوێت لەسەر ئینتەرنێت
  4. دەتوانیت هەروەها لەسەر Netlify هۆست بکەیت بە ڕاکێشانی فایلەکە بۆ سەر ماڵپەڕەکەیان

پەیوەندی بکە بەم ماڵپەڕانە: GitHub Pages, Netlify, Vercel
